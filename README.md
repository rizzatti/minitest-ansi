# MiniTest::Ansi

Colorize your minitest output using ANSI colors.

## Installation

Add this line to your application's Gemfile:

    gem 'minitest-ansi'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install minitest-ansi

## Usage

    require 'minitest/ansi'
    MiniTest::Ansi.use!
